## なにこれ

Linux（Wayland / GNOME）で、**OpenAIのAPIキーを登録するだけで使える音声入力ツール**を作りました。ほぼキーボードを使わずに、喋った内容をいま開いてるアプリにそのまま入れられます。

- 指定キーを押してる間だけ録音して、離すと変換
- OpenAIの `gpt-4o-transcribe` で変換（句読点もいい感じに付く。日本語もそのまま高精度、`language`設定で他の言語もいける）
- 変換した文字を、いま開いてるアプリ（Claude / ブラウザ / ターミナルとか）のカーソル位置にそのまま入れてくれる
- 変換履歴を一覧で見れて、ボタンでコピーできるGUIも付けた

ローカルにモデル置いたりの面倒なセットアップは無し。APIキー1個入れれば動きます。

リポジトリはこれ。MITなので、使いたい人は勝手に使ってくださいー。

https://github.com/usar00/voicein

ちなみにこの記事も、ほぼこのツールで喋って書いてます。

## なんで作ったの？

音声入力って便利なんだけど、いい感じのやつってだいたい有料サブスクなんだよね。月額で課金がずっと続くし、そもそもLinux対応が弱い。Macは充実してるのに、Linuxだと選択肢が一気に減る。

かといってローカルにモデル（Whisperとか）を立てるのは、GPUとかセットアップとかで地味に面倒。もっとこう、APIキーをペッと登録するだけで動くやつが欲しかった。

OpenAIの音声認識APIって従量課金で激安で、`gpt-4o-transcribe` だと音声1分でだいたい1円くらい。月にどれだけ喋っても数十円〜数百円で収まる。サブスクの月額を払い続けるより、自分でAPI叩いたほうが安い。精度も普通に高いし、日本語もそのままいける。

要するに、高いから・面倒だから、APIキー登録だけで動くやつを自分で作った、ってだけです。

## 仕組み

そんなに難しいことはしてなくて、わりと素朴です。

```
[キー押してる間] parecord で録音(16kHz mono WAV)
     │  キーを離す
     ▼
OpenAI gpt-4o-transcribe で日本語に変換
     │
     ▼
wl-copy でクリップボードに入れて → 仮想キーボード(evdev/uinput)で
Ctrl+V を送る → カーソル位置に貼り付け
     │
     ▼
履歴(JSONL)に保存 → GUIで一覧・コピー
```

ポイントだけ。

### 文字入れは「クリップボード貼り付け」でやる

日本語を1文字ずつキーコードでタイプするのは（キー配列依存で）しんどいので、テキストをクリップボードに入れて Ctrl+V を送る方式にしました。これなら日本語でも絵文字でも確実に入る。

### Wayland でも X11 でも動く

GNOMEのWaylandって合成入力に厳しいんだけど、カーネルレベルの `uinput`（仮想キーボード）と `evdev`（キー検出）を使ってるので、Wayland / X11 どっちでも動きます。`/dev/uinput` と `/dev/input` を `input` グループで触れるようにするのがコツ。

### キーごとに貼り付け方を変えられる

ターミナルって貼り付けが `Ctrl+Shift+V` だったりするので、キーごとに割り当てを変えられるようにしてます。あと、押した瞬間に反応するとピコピコうるさいので、0.15秒くらいの「溜め」を入れて、ちょっと押し続けたら録音開始するようにしてます。

```ini
[bindings]
KEY_RIGHTCTRL = ctrl+v          ; 普通のアプリ(Claude / ブラウザ / 入力欄)
KEY_RIGHTSHIFT = ctrl+shift+v   ; ターミナル
```

## 使い方

```bash
git clone https://github.com/usar00/voicein.git
cd voicein
bash install.sh        # 依存入れて、権限設定して、venv作る
# 一度ログアウト/ログイン(inputグループ反映のため)
# config.ini に OpenAI APIキーを入れる
./run.sh               # 起動
```

あとは右Ctrlを押しながら喋って離すだけ。カーソル位置に日本語が入ります。

```bash
./history.sh           # 変換履歴のビューア(一覧 + コピー)
```

ログイン時の自動起動（systemd user service）も同梱してます。

## 動く環境

- Linux（**Wayland / GNOME / Ubuntu** で作って確認。X11でも動く作り）
- 音声は PulseAudio か PipeWire（`parecord`）
- OpenAI APIキー（従量課金。`gpt-4o-transcribe` で約1円/分）

他のディストロでも、パッケージ名を読み替えればだいたい動くと思います。

## ハマったとこ（OpenAIの課金）

ひとつだけ注意。OpenAIは前払い式なので、APIキー作っただけだと `insufficient_quota` で動きません。クレジットの購入が必要。

で、地味な罠なんだけど、**OpenAIはJCBカードを弾きます**。自分もJCBで決済失敗して、Mastercard（デビットでOK）に変えたら通った。同じとこで詰まったらカード変えてみてください。
