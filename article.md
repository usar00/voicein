## なにこれ

Linux（Wayland / GNOME）向けに、**OpenAIのAPIキーを登録するだけで使える音声入力ツール**を作りました。ほぼキーボードを使わずに、話した内容をそのまま、いま開いているアプリへ入力できます。

- 指定したキーを押している間だけ録音し、離すと変換します
- OpenAIの `gpt-4o-transcribe` で変換します（句読点も自然に付きます。日本語も高い精度で、`language` 設定で他の言語にも対応できます）
- 変換したテキストを、いま開いているアプリ（Claude / ブラウザ / ターミナルなど）のカーソル位置へ自動で挿入します
- 変換履歴を一覧で見られて、ボタンひとつでコピーできるGUIも付けています

ローカルにモデルを用意するような手間はいりません。APIキーをひとつ登録すれば動きます。

リポジトリはこちらです。MITライセンスですので、どなたでも自由に使っていただけます。

https://github.com/usar00/voicein

ちなみにこの記事も、ほとんどこのツールで話しながら書いています。

## なぜ作ったのか

音声入力は、慣れるととても快適です。手を止めずに文章を書けるので、一度使うと手放せなくなります。

ただ、完成度の高いものの多くは月額のサブスクリプションで提供されていて、使い続けるにはそれなりのコストがかかります。一方で、ローカルにモデル（Whisperなど）を立てる方法は無料に近いものの、環境構築やGPUの用意など、最初のハードルが少し高めです。

そこで、**「APIキーを登録するだけで、すぐに使い始められる」**くらいの手軽さで、Linux上で動く音声入力が欲しくなりました。常に持ち歩く作業環境がLinuxなので、ここで気持ちよく音声入力ができると、日々の体験がかなり変わります。

幸い、いまは音声認識のAPIがとても手の届きやすい価格になっています。`gpt-4o-transcribe` であれば音声1分あたり約1円ほど。これくらいなら、必要なときに必要なだけ使う、という形が無理なく成り立ちます。精度も実用十分で、日本語もそのまま自然に書き起こせます。

個人が、自分の使いたい道具を、自分の手で組み立てられる。そういうことが現実的にできる時代になったのだと、作ってみて改めて感じました。同じように「手軽に音声入力したい」と思っている方の出発点になれば嬉しいです。

## 仕組み

特別なことはしておらず、構成はかなりシンプルです。

```
[キーを押している間] parecord で録音(16kHz mono WAV)
     │  キーを離す
     ▼
OpenAI gpt-4o-transcribe で変換
     │
     ▼
wl-copy でクリップボードへ → 仮想キーボード(evdev/uinput)で
Ctrl+V を送出 → カーソル位置に貼り付け
     │
     ▼
履歴(JSONL)に保存 → GUIで一覧・コピー
```

要点を3つだけ補足します。

### 文字の入力は「クリップボード経由の貼り付け」で行う

日本語を1文字ずつキーコードで送るのはキー配列依存で扱いづらいため、テキストをクリップボードに入れて `Ctrl+V` を送る方式にしました。これなら日本語でも絵文字でも、確実に入力できます。

### Wayland でも X11 でも動く

GNOMEのWaylandは合成入力に対して制約が強いのですが、カーネルレベルの `uinput`（仮想キーボード）と `evdev`（キー検出）を使うことで、Wayland / X11 のどちらでも動作します。`/dev/uinput` と `/dev/input` を `input` グループで扱えるようにするのがポイントです。

### キーごとに貼り付け方法を変えられる

ターミナルでは貼り付けが `Ctrl+Shift+V` のことがあるので、キーごとに割り当てを変えられるようにしています。また、押した瞬間に反応すると操作感が忙しないため、0.15秒ほどの「溜め」を入れて、少し押し続けたら録音を開始するようにしています。

```ini
[bindings]
KEY_RIGHTCTRL = ctrl+v          ; 通常のアプリ(Claude / ブラウザ / 入力欄)
KEY_RIGHTSHIFT = ctrl+shift+v   ; ターミナル
```

## 使い方

```bash
git clone https://github.com/usar00/voicein.git
cd voicein
bash install.sh        # 依存の導入・権限設定・venv作成
# 一度ログアウト/ログイン(inputグループ反映のため)
# config.ini に OpenAI APIキーを設定
./run.sh               # 起動
```

あとは右Ctrlを押しながら話し、離すだけです。カーソル位置に文字が入ります。

```bash
./history.sh           # 変換履歴ビューア(一覧・コピー)
```

ログイン時の自動起動（systemd user service）も同梱しています。

## 動作環境

- Linux（**Wayland / GNOME / Ubuntu** で開発・確認しています。X11でも動作する設計です）
- 音声: PulseAudio または PipeWire（`parecord`）
- OpenAI APIキー（従量課金。`gpt-4o-transcribe` で約 $0.006/分）

`evdev`/`uinput` などLinux固有の仕組みを使っているため、現状はLinux向けです。他ディストロでも、パッケージ名を読み替えればおおむね動作します。

## 補足：OpenAIの課金について

ひとつだけ事前の注意点です。OpenAIは前払い式のため、APIキーを作成しただけでは `insufficient_quota` となり動きません。先にクレジットの購入が必要です。

また、決済時に **JCBカードは通らないことが多い**です。私自身もJCBで失敗し、Visa/Mastercard（デビットでも可）に変更したところ通りました。もし同じところで止まったら、カードを変えてみてください。
